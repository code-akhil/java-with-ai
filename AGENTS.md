# Project Instructions

- Prefer explaining the plan before editing files unless I explicitly ask you to implement.
- Keep Java code beginner-friendly, readable, and close to the existing style.
- This is a Spring Boot project built with Maven, and most changes should be validated with Maven commands.
- Avoid large refactors unless they are necessary for the requested change.
- Do not change unrelated behavior while working on a specific feature.
- After code changes, compile with:

```bash
mvn clean test-compile
```

- Run the application with:

```bash
mvn spring-boot:run
```

- If Maven is not installed, install it before running or testing the project.
